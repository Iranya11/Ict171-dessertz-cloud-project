01 — Project Description

Connecting culture through sweetness

Dessertz is a cloud-based web platform dedicated to celebrating the extraordinary diversity of desserts, pastries, and sweet culinary traditions from cultures around the world. Every dessert carries within it a story of heritage, community, and identity — this platform brings those stories together in one accessible, beautifully presented digital space.

The platform is hosted on Microsoft Azure, deployed on an Ubuntu 22.04 LTS virtual machine using the Apache HTTP Server, following the Infrastructure as a Service (IaaS) model. What began as a static concept page is now a working front end: browse desserts by region, search a real recipe library with scalable serving sizes, and read the cultural stories behind each dish — all running client-side, ready to be connected to a database and accounts as the next milestone.
